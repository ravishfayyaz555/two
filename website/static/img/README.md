## Image Guidelines

- **Format**: PNG or JPG
- **Max size**: 800px width for optimal mobile rendering
- **Alt text**: Always include descriptive alt text in markdown
- **Compression**: Optimize images for web (use tools like TinyPNG)

## Usage in Markdown

```markdown
![Descriptive alt text](/img/ch1-physical-ai-overview.png)
```
